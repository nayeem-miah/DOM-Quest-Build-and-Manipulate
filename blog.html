What is the DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as a tree of nodes.

DOM Tree Structure

Every HTML element becomes a node in the DOM tree. For example: 

<html>
<body>
  <h1>Title</h1>
  <p>Paragraph</p>
</body>
</html>

This structure is represented in memory as a tree of objects.

Common DOM Methods

  • document.getElementById('id')
  • document.getElementsByClassName('class')
  • document.querySelector('selector')
  • element.innerHTML ‚Äì Get or set HTML inside an element
  • element.addEventListener('event', function)
